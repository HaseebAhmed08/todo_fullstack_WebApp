# NextJS Development Skill

## Overview
NextJS is a popular React-based framework for building production-ready web applications. It provides features like server-side rendering, static site generation, routing, API routes, and more.

## Key Features
- File-based routing system using the `pages` or `app` directory
- Server-Side Rendering (SSR) and Static Site Generation (SSG)
- API routes for backend functionality
- Built-in CSS and Sass support
- Automatic code splitting
- Fast Refresh for improved development experience

## Best Practices
- Use dynamic imports for code splitting
- Leverage NextJS Image component for optimization
- Implement proper error handling with custom error pages
- Use environment variables for configuration
- Follow SEO best practices with Head component
- Optimize performance with lazy loading

## Common Commands
- `npm run dev` - Start development server
- `npm run build` - Build for production
- `npm run start` - Start production server

## File Structure
- `pages/` - Contains route components
- `public/` - Static assets
- `components/` - Reusable UI components
- `styles/` - Global stylesheets
- `api/` - API routes (if using pages router)

This skill provides knowledge about NextJS development for building modern web applications.
